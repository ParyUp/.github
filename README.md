# ParyUp

ParyUp is a mobile application that combines event discovery, ticket purchasing, and social experience in a single platform.

The idea is simple: decide to go out in seconds, see what’s nearby, who’s going, and get in with no friction.

---

## What is ParyUp

ParyUp connects users with clubs, bars, and events in real time.

It’s not just a list of parties. It’s a social layer on top of nightlife.

* Discover nearby events filtered by date, time, and location
* See what’s happening right now, not just what’s coming up
* Buy tickets directly in the app
* Interact with other attendees
* Access content from clubs and promoters

---

## Core Features

### Smart discovery

Events organized by:

* Today, tomorrow, weekend
* Near you
* Recommended

### Event view

Each event includes:

* Clear information on when and where
* Strong visual cover
* Attendees and activity
* Direct ticket purchase

### Social layer

* Social-style user profiles
* Stories and ephemeral content
* Chats and groups
* Post-event connections

### For clubs

* Event creation and management
* Direct channel with their audience
* Promotion and visibility
* CRM-like tools (in development)
